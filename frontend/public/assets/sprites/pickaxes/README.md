# Pickaxe Sprites Directory

## Place Your Pickaxe Sprites Here!

This directory should contain 5 pickaxe sprite files:

### Required Files

```
pickaxe-wooden.png      ← Tier 0 (Free starter)
pickaxe-iron.png        ← Tier 1 (100 GLD)
pickaxe-steel.png       ← Tier 2 (300 GLD)
pickaxe-mythril.png     ← Tier 3 (1000 GLD)
pickaxe-adamantite.png  ← Tier 4 (5000 GLD)
```

### Sprite Specs

- **Format**: PNG with transparency
- **Size**: 16x16 or 32x32 pixels
- **Style**: Pixel art
- **Orientation**: Diagonal/downward facing

### Colors

- **Wooden**: Brown handle + gray/stone head
- **Iron**: Dark handle + silver head
- **Steel**: Dark handle + blue-steel head
- **Mythril**: Ornate + purple/magical glow
- **Adamantite**: Legendary + orange-red glow

---

## Don't Have Sprites Yet?

**No problem!** The game auto-generates fallback textures if sprites are missing.

Just drop your PNG files here when ready - they'll be picked up automatically!

---

## Testing

1. Add sprite files to this directory
2. Refresh your browser
3. Pickaxes will appear in-game!
